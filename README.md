#### 💡 Innovation Highlights

<table>
  <tr>
    <td width="50%">
      <h4>🧠 Deep Learning Engine</h4>
      <ul>
        <li><b>MT-STNet</b> (Multi-Task Spatio-Temporal Network)</li>
        <li>Graph Convolutional Networks with highway topology</li>
        <li>Spatio-temporal attention mechanisms</li>
        <li>Generative inference to prevent error accumulation</li>
      </ul>
    </td>
    <td width="50%">
      <h4>🌊 Physics-Based Engine</h4>
      <ul>
        <li><b>LWR Traffic Flow Theory</b> integration</li>
        <li>Shockwave detection using Rankine-Hugoniot conditions</li>
        <li>Real-time congestion propagation prediction</li>
        <li>Precise arrival time estimation</li>
      </ul>
    </td>
  </tr>
</table>

### 📊 Project 2: Online Behavior & Cyberbullying Research

> **🏆 Best Popularity Award + 🎖️ Merit Award** - Academia Sinica Data Science Stroll 2024

<p align="center">
  <a href="https://github.com/timwei0801/MVA-Internet-use-and-bullying">
    <img src="https://img.shields.io/badge/📂_Source_Code-GitHub-181717?style=for-the-badge&logo=github"/>
  </a>
</p>

#### 💡 Research Innovation

<table>
  <tr>
    <td width="50%">
      <h4>🔬 Methodology Breakthrough</h4>
      <ul>
        <li><b>First application</b> of GAP (Generalized Association Plots) in online behavior research</li>
        <li>Novel integration of 5 multivariate analysis methods</li>
        <li>Complete analytical chain from exploration to prediction</li>
      </ul>
    </td>
    <td width="50%">
      <h4>📈 Key Discoveries</h4>
      <ul>
        <li>Identified <b>5 distinct user clusters</b> via GAP analysis</li>
        <li>Discovered <b>4 key behavioral factors</b></li>
        <li>Novel finding: <b>Conflict Tolerance</b> as new factor</li>
      </ul>
    </td>
  </tr>
</table>

#### 📋 Research Details

| Aspect | Detail |
|:---|:---|
| **Data Source** | Taiwan Communication Survey (2021) |
| **Sample Size** | N = 672 participants |
| **Age Range** | 19-61 years old |
| **Methods** | GAP, PCA, Factor Analysis, CCA |
| **Tools** | Python, R, Statistical Modeling |

#### 🎯 Impact

- Established a **new theoretical framework** for cyberbullying behavior analysis
- Provided **paradigm reference** for future online behavior studies
- Successfully communicated complex statistical findings to **non-technical audiences**
